• Определение блочной модели тегов
    1. №1 Элемент "p" относится к блочным элементам, но он не должен содержать внутри себя другой элемент "p" а также любой другой блочный элемент.
    2. №2 Теги b относятся к устаревшей блочной модели HTML и используются для задания жирного шрифта тексту.
    3. №3 Тег h2 в HTML используется для создания заголовков второго уровня.
    4. №4 Тег h3 в HTML используется для создания заголовков тре уровня.
    5. №5 Теги i относятся к инлайновой модели блоков.
    6. №6 Теги p относятся к блочной модели блоков.
    7. №7 Теги a относятся к инлайновой модели блоков.
